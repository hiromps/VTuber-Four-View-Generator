# VTuber四面図ジェネレーター - ランディングページ

このディレクトリには、VTuber四面図ジェネレーターのランディングページが含まれています。

## ファイル構成

```
public/landing/
├── index.html      # メインのHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptファイル
└── README.md       # このファイル
```

## アクセス方法

### 開発環境
ローカルサーバーを起動後、以下のURLでアクセスできます：
- `http://localhost:3000/landing/`
- `http://localhost:3000/landing/index.html`

### 本番環境
- `https://yourdomain.com/landing/`

## 主な機能

### 1. レスポンシブデザイン
- デスクトップ、タブレット、モバイルに完全対応
- ブレークポイント：
  - モバイル: 〜768px
  - タブレット: 769px〜992px
  - デスクトップ: 993px〜

### 2. インタラクティブ要素
- モバイルメニューの開閉
- FAQアコーディオン
- スムーススクロール
- スクロール時のヘッダー影追加

### 3. デザインシステム

#### カラーパレット
- **メインカラー**: `#FF9EC2` (パステルピンク)
- **アクセント**: `#9ECFFF` (パステル水色)
- **背景**: `#FEFEFE` (オフホワイト)
- **テキスト**: `#4A4A4A` (ダークグレー)

#### フォント
- **Noto Sans JP** (Google Fonts)
- ウェイト: 400, 500, 700, 900

## カスタマイズ方法

### 色の変更
`styles.css`の`:root`セクションで色変数を変更できます：

```css
:root {
    --color-primary: #FF9EC2;     /* メインカラー */
    --color-accent: #9ECFFF;      /* アクセントカラー */
    /* その他の変数... */
}
```

### コンテンツの編集
`index.html`の各セクションを編集してください：

- **ヒーローセクション**: `.hero`
- **特徴セクション**: `.features`
- **使い方セクション**: `.how-to-use`
- **FAQ**: `.faq`
- **フッター**: `.footer`

### アニメーションの調整
`styles.css`のアニメーションセクションで調整可能：

```css
@keyframes fadeInUp {
    /* アニメーション定義 */
}
```
