# ヒーロー画像の配置について

フロントページのヒーローセクションに画像を表示するには、以下の手順で画像を配置してください：

## 画像配置手順

1. 提供されたVTuber画像を `hero-vtuber.png` という名前で保存します
2. このpublicフォルダに `hero-vtuber.png` を配置します

## 推奨画像仕様

- 形式: PNG（透過背景推奨）
- サイズ: 800x600px 以上
- ファイル名: `hero-vtuber.png`

## 表示確認

画像を配置後、開発サーバーで確認してください：

```bash
npm run dev
```

ブラウザで http://localhost:3000 にアクセスすると、ゲーミフィケーションデザインのフロントページが表示されます。
